# 📊 Dashboard de Redes Sociais - Política

Dashboard interativo em Python (Dash + Plotly) para análise de desempenho de candidatos políticos em redes sociais.

---

## 🚀 Funcionalidades

- 📊 Seguidores, interações e engajamento por perfil
- 🔀 Comparação entre redes sociais
- 📈 Análise de eficiência (seguidores vs interações)
- 📝 Quantidade de posts por candidato
- ☁️ Nuvens de palavras por candidato

---

## 🗂️ Estrutura

```
app.py
pages/
├── geral.py
├── comparativo.py
└── posts.py
data/
├── vereadores/
└── prefeitos/
data_loader.py
requirements.txt
```
---

## ⚙️ Instalação

```bash
pip install -r requirements.txt
```

---
## ▶️ Execução
```bash
python app.py
```
